# Discord banners

## 🚀 Overview

This is the frontend for [Discord banners](https://discord-banners.terminaate.site), an open-source project that
provides banner image for your discord profile.

## 📦 Tech Stack

- React
- Vite
- React-Router-Dom
- Framer-motion
- Axios
